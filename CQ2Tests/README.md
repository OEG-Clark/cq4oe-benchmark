# CQ2Tests


### Task Definition

Given a set of requirements (CQ) as the input, and use the ontology data as external knowledge, the task is to generate the tests that is needed for validating requirement for the ontology.

### Dataset

- [Auroral Core Ontology](https://auroral.iot.linkeddata.es/def/core): This ontology aims at modelling the data related to users, items, notification and organisations for the AURORAL project.

### Examples and Temp Result

- Input: A notification can be read
- Output: Notification subclassOf isUnread only boolean
