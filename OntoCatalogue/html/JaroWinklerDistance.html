JaroWinklerDistance: String similarity metric emphasizing common prefixes

This metric computes the Jaro-Winkler similarity between a generated label and each gold-standard label, returning a score in [0, 1]. The Jaro-Winkler variant boosts matches that share a common prefix, which is useful for terminology that follows shared roots.

Calculation Process:

  1. Compute the Jaro similarity between the generated label and each gold label, accounting for matching characters and transpositions within a sliding window.
  2. Apply the Winkler boost: scale up the score for pairs sharing a prefix of up to four characters.
  3. Take the maximum similarity across all gold labels.
  4. A pair is considered a match if the score is greater than or equal to 0.8 (default lexical threshold).

Example:

Generate_Label: WineColor

Gold_Standard_Label: WineColour

JaroWinklerDistance(Generate_Label, Gold_Standard_Label) = 0.96

What is being measured?

This metric tolerates small character-level differences, especially in suffixes (e.g. British vs American spelling, plural forms, minor typos), while penalizing changes early in the string.
