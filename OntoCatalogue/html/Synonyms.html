Synonyms: Exact match against WordNet synonym relations

This metric evaluates whether the generated label, expanded with its WordNet synonyms, exactly matches any label in the gold-standard ontology.

Calculation Process:

  1. Retrieve all synonyms of the generated label from WordNet, forming a generated-label set.
  2. Compare every member of the generated-label set against every gold-standard label using exact (case-insensitive) match.
  3. If at least one pair matches, the result is 1; otherwise the result is 0.

Example:

Generated_label: Wine

Generated_label_set: Wine, alcohol, vintage, red_wine, white_wine, dark_red, …

Gold_standard_label: red_wine

Result: Synonyms(Gold_standard_label, Generated_label_set) = 1

What is being measured?

This metric identifies semantically equivalent concepts expressed through different lexical forms by leveraging the synonym relation curated in WordNet (or a comparable lexical dictionary).
