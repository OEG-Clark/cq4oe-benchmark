SequenceMatch: Token-level string similarity comparison

This metric computes a token-level similarity between a generated label and each gold-standard label using the longest contiguous matching subsequence (the same algorithm Python's difflib.SequenceMatcher uses), returning a score in [0, 1].

Calculation Process:

  1. Tokenize each label into characters (or sub-tokens after CamelCase split and underscore normalization).
  2. Find the longest contiguous matching block, then recursively process the unmatched left and right sides.
  3. Return ratio = 2 * matches / (len(generated) + len(gold)).
  4. A pair is considered a match if the score is greater than or equal to 0.8 (default lexical threshold).

Example:

Generate_Label: WineProducer

Gold_Standard_Label: ProducerOfWine

SequenceMatch(Generate_Label, Gold_Standard_Label) = 0.85

What is being measured?

This metric rewards labels that share long contiguous substrings, even when the prefix differs. It is well suited to detecting compound terms reordered or partially renamed.
